字典详情页面可通过字典数据页面的 “新增” 和 “编辑” 按钮进入

字段描述
1. 类别编码：可以编辑，长度限制在32以内；

2. 类别名称：可以编辑，长度限制在32以内；

3. 创建时间：不可编辑，只用于显示，如果通过 “新增” 按钮进入该页面则不需要显示；
4. 更新时间：不可编辑，只用于显示，如果通过 “新增” 按钮进入该页面则不需要显示；


列表
1. 某一字典类别下的所有字典项；
2. 按创建时间升序排序；

操作
1. “保存” 按钮触发后，将数据提交至后台保存变更；
2. “新增字典项” 按钮触发后，弹出字典项编辑页，用于新增字典项。保存后同步变更列表数据。
3. “编辑” 按钮触发后，弹出字典项编辑页；保存后同步变更列表数据。
4. “删除” 按钮触发后，需弹出确认删除按钮，后端需要校验字典项是否被引用，如果被引用则无法删除。


字典项编辑页

1. 字典类别编码和字典类别名称不可编辑
2. 字典项编码必填、长度限制32
3. 字典项名称必填、长度限制64
